## Why

The N2E tracker records cartridge stat rolls per character but has no concept of which named cartridge set is equipped. Without a catalog identity, users cannot express which set they are farming toward, and the scoring system cannot reflect the most impactful build decision (set choice) — only the stat roll quality.

## What Changes

- Add `ALL_CARTRIDGES` static catalog (12 named sets × 3 rarities) to `src/data/neverness-to-everness/cartridges.ts`, auto-generated by the existing weekly update script via the `shards` GraphQL query
- Add `cartridge_id TEXT` column to `n2e_tracked_characters` DB table
- Add `cartridgeId: string | null` field to `N2ETrackedCharacter` type and `N2ECharacterPatch`
- Add `cartridgeId: string | null` field inside `cartridgePreferences` on `N2ETrackedCharacter`
- Update `CartridgeEditorModal` equip tab: name picker (dropdown of 12 set names) → rarity picker (B/A/S) replaces the current single rarity dropdown
- Update `CartridgeEditorModal` preferences tab: add single named-cartridge preference field
- Revise scoring weights: cartridgeId match (0.35) + main stat (0.30) + sub stats (0.35); cartridgeId match is rarity-aware (same set + same/better rarity = 1.0; one rarity below = 0.6; two rarities below = 0.3; wrong set = 0.0)
- Extend weekly update script to generate `cartridges.ts` from the `shards` query (no image pipeline — name + rarity badge only)

## Capabilities

### New Capabilities

- `n2e-cartridge-catalog`: Static catalog of named N2E cartridge sets sourced from the `shards` GraphQL endpoint, generated by the weekly update script.

### Modified Capabilities

- `n2e-character-detail`: Cartridge equipment requirement gains a `cartridgeId` field (named set identity); cartridge preferences gain a `cartridgeId` preference field. Picker UX changes to name-first → rarity-second.
- `n2e-cartridge-scoring`: Score formula revised to three-term weighted sum (cartridgeId 0.35 + mainStat 0.30 + subStats 0.35) with rarity-aware partial credit for the cartridgeId term. Weights for main stat and sub stats change from 0.4/0.6.
